# GOSR™ (CCBP Runtime Stack)

**GOSR™ — Game-play Opportunistic System Recognition** is a CCBP-aligned, CLP-governed, friction-first ranking engine for identifying emerging niche opportunities with minimal drift and maximum legibility.

This repo contains a **ChatGPT-first “constitutional prompt stack”**:

- Kernel (runtime rules)
- Output Contract (structured, comparable outputs)
- Patch Protocol (controlled evolution)
- Archon Failure Modes (quiet diagnostics + anomaly-ready logging)

## Canonical Identity (prevents acronym collisions)

- **System Name:** GOSR™ — Game-play Opportunistic System Recognition  
- **Canonical System Name (CSN):** `gosr.gameplay-opportunistic-system-recognition.ccbp`

If identity cannot be confirmed, the runtime must request clarification before execution.

## Quickstart (ChatGPT Runtime)

1) Paste the Kernel prompt into a new ChatGPT thread (or Custom GPT instructions):
   - `modules/GOSR/GOSR_KERNEL_v0.2.prompt`

2) Run an evaluation using:

```text
RUN gosr.gameplay-opportunistic-system-recognition.ccbp v0.2
Target niche: ...
Region: ...
Time horizon: ...
Budget band: bootstrap / small / medium / capital-backed
Constraints: ...
```
